# 商品图片目录

请将商品图片放在此目录下，文件名需与商品名称对应。

## 图片命名规则

图片文件名 = 商品名称 + .jpg

例如：
- iPhone 15 Pro.jpg
- MacBook Pro 14.jpg
- AirPods Pro 2.jpg
- 华为Mate 60 Pro.jpg
- 小米14 Ultra.jpg
- 男士休闲夹克.jpg
- 女士连衣裙.jpg
- 运动T恤.jpg
- 牛仔裤.jpg
- 进口坚果礼盒.jpg
- 有机牛奶.jpg
- 精品咖啡豆.jpg
- 北欧风沙发.jpg
- 智能台灯.jpg
- 不锈钢锅具套装.jpg

## 图片规格建议

- 尺寸：800x800 像素（正方形）
- 格式：JPG 或 PNG
- 大小：建议不超过 500KB

## 默认图片

如果商品没有对应图片，系统会显示 default.jpg
